1. BIOS / UEFI

Sobald der Computer eingeschaltet wird, wird als erstes Das BIOS (In neueren Modellen UEFI) geladen und ausgeführt. Dieses Initialisiert alle nötigen Hardwarekomponenten.

Das UEFI besitzt einen integrierten Boot-Manager, welcher dem Benutzter die möglichkeit, bietet der, Startgerät auszuwählen sowie den Start in das BIOS.

2. POST (Power On Self Test)

Hier führt das BIOS eine reihe an Test durch, um Sicher zu stellen, dass die Hardwarekomontent ordnungsgemäß funktionieren.

3. Boot Sequenz

Das Bios sucht nach startfähige Geräte(HDD, SSD, M2, ...) und führt das erste in der Startreihenfolge aus. Dann lädt das BIOS den MBR und führt den enthaltenden Bootloader aus.

Das UEFI enthält auch eine Secure-Boot Funktion, welche überprüft, ob das Betriebssystem vor dem Booten manipuliert wurde.

4. MBR (Master Boot Record)

Der Bootloader im MBR ist zuständig für das Laden des restlichen Betriebssystems wie z.b. den Kernel.

5. Betriebssystem

Sobald der Bootloader das Betriebssystem geladen hat, gibt es die Kontrolle and dieses ab. Der Kernel lädt nun Treiber und initialisiert Hardware- sowie Softwarekomponenten.
